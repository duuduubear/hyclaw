## 项目简介

基于 NVIDIA Jetson Nano 的**插电即用**边缘AI运维节点方案。聚焦量产落地痛点，完成出厂预置配置、mDNS自动发现、Windows/macOS轻量客户端、加密SSH远程运维全链路闭环。

- **论文**：[论文.md](论文.md) （完整5080字技术文档）
- **客户端下载**：https://www.chenhuyin.com/hyclawdownload.html

## 核心特性

- 标准化出厂环境固化，一键部署
- 原生轻量跨平台客户端（<20MB）
- 加密SSH + mDNS局域网自动发现
- 本地闭环存储，注重数据安全
- 支持GPIO/UART外设拓展
